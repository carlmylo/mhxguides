This is a quick setup guide to get REAPER up and running without having to hunt down lots of files from various places.  
All of the included resources are confirmed working and made for the latest version of REAPER and Python on Windows and Linux.

<Aside>This will NOT teach you how to chart!</Aside>
If you need help with that, here are some resources:

### Written guides
If you're into reading at your own pace, here's some outstanding written guides:
* <a href="http://docs.c3universe.com/rbndocs/index.php?title=Authoring" target="_blank" rel="noopener noreferrer">[RBN/C3 Authoring Documentation]</a> is old but is extremely reliable and well written. It has stood the test of time and most people still rely on it. It was originally written by Harmonix but this version also includes some extra advice from the C3 charting group.
  * <a href="https://web.archive.org/web/20250417062932/http://docs.c3universe.com/rbndocs/index.php?title=Authoring" target="_blank" rel="noopener noreferrer">[Wayback version in case the main link is down].</a>

* There is also <a href="https://rhythmgamingworld.com/rock-band-3-customs-book" target="_blank" rel="noopener noreferrer">[an amazing book by doa]</a> but keep in mind that certain bits of it suggest installing the outdated version of REAPER **for the moment**. You can and should ignore this until the new version comes out.

* <a href="https://therogerland.tumblr.com/proguide" target="_blank" rel="noopener noreferrer">[Pro Guitar Guide]</a> - You're probably never gonna touch Pro Guitar/Bass, but for the masochists, this is the only good guide out there.

### Video guides
If you prefer videos, there's a few longform videos that can help:
* Broforce released a <a href="https://www.youtube.com/playlist?list=PLrIDLbHWXFJcidr8R95lH_gNfWpC-6hFh" target="_blank" rel="noopener noreferrer">[playlist of charting videos]</a> that use this guide as the install base. Not only that but different instruments are split into different videos and the videos themselves have timestamp chapters.

* For vocals, <a href="https://www.youtube.com/@addymillscustoms" target="_blank" rel="noopener noreferrer">[AddyMills]</a> (a longtime charter with a great reputation) has an outstanding video guide that uses some features exclusive to the latest versions of REAPER.  
<a href="https://youtu.be/Vlq8NI2KOhg" target="_blank" rel="noopener noreferrer">[Part 1]</a>  <a href="https://youtu.be/whh5BYPx7Fs" target="_blank" rel="noopener noreferrer">[Part 2]</a>

* On the older side, there is <a href="https://www.youtube.com/watch?v=FHeGkA4nv4g" target="_blank" rel="noopener noreferrer">[a very thorough video by Truce]</a> but you should keep in mind that it may mention that you should install an old version of REAPER. You can and should ignore this.

If you have questions or get stuck, you can ask for help in the <a href="https://discord.gg/milohax" target="_blank" rel="noopener noreferrer">[MiloHax Discord]</a>. There’s even a <a href="https://discord.com/channels/961352072140324924/961395145809465405" target="_blank" rel="noopener noreferrer">[dedicated charting channel]</a>.
