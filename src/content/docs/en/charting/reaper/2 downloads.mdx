## Installing REAPER
<Steps>
1. <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer">[Download and install the latest 64-bit version of REAPER.]</a>.
    * **Please DO NOT enable the `Portable install` option when installing!**  
    > <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer"><Image src={webreaper} alt="REAPER's download page cropped to show the download button for Windows. The cursor is over the green Download button in the Windows section." /></a>

2. <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer">[Download and install the latest 64-bit version of SWS / S&M Extension.]</a>.
    * Please **keep `Python ReaScript support` enabled when installing!**  
    > <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer"><Image src={websws} alt="SWS/S&M Extension's download page, cropped to show the download button for Windows. The cursor is over the 'Download x64'  button in the Windows section." /></a>

3. <a href="https://www.python.org/downloads" target="_blank" rel="noopener noreferrer">[Download and install the latest version of Python 3 if you don't have it already.]</a>.
    * **Do NOT download Python install manager!**
    * **Python 2 is NOT the same and will not work**.
    > <a href="https://www.python.org/downloads" target="_blank" rel="noopener noreferrer"><Image src={webpython} alt="Python's download page cropped to show a cursor over 'get the standalone installer for Python 3.14.2'." /></a>
4. <a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer">[Click here to download the REAPER Files `.zip` archive.]</a>.
    * a. Extract the `.zip` archive somewhere convenient.
    * b. Drag the `Common` folder into `DRAG 'Common' HERE` shortcut.
    * c. Drag `REAPER` into the `DRAG 'REAPER' HERE` shortcut.  
    * d. Replace any files if you're asked.
    > <a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer"><Image src={explorerdrag} alt="Folders being dragged into shortcuts to other folders." /></a>
</Steps>
