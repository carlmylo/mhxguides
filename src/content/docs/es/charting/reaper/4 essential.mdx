## Atajos importantes
Los siguiente atajos fueron agregados:
* F6 - <a href="https://github.com/RaiderGG/EVENTS-Tools" target="_blank" rel="noopener noreferrer">[EVENTS Tools]</a>: Herramientas para la pista de EVENTS. Ayuda con estados de publico y eventos de sección.  
  > <Image src={eventstools} alt="EVENTS Tools, mostrando utilidades para convertir marcadores a secciones, marcar donde inicia y termina la canción y varios eventos de publico." />
* F7 - <a href="https://github.com/tposejank/EncoreVocalPreview/" target="_blank" rel="noopener noreferrer">[EncoreVoxPreview]</a>: Visualiza tu chart de vocales. ¡Incluye armonías! Cambia entre pistas con las teclas de `[` y `]`.   
Ciertos teclados (como los de LATAM) usan `SHIFT` + `{` y `SHIFT` + `}`.  
  > <Image src={encorevox} alt="Encore Vox Preview mostrando tres partes de vocales. Tienen letras distintas y están separadas por color." />
* F8 - <a href="https://github.com/Hoph2o/HOPCAT/" target="_blank" rel="noopener noreferrer">[HOPCAT]</a>: Una versión actualizada de CAT. Esto es una utilidad que hace demasiadas cosa y ayuda con varias cosas tediosas.
  > <Image src={hopcat} alt="HOPCAT mostrando varias utilidades como animaciones automáticas, reducciones y muchas otras cosas." />
* F9 - <a href="https://github.com/Deevun155/venuegen" target="_blank" rel="noopener noreferrer">[VenueGen]</a>: Ya no hagas tu Venues con eventos latosos. ¡Usa notas!  
  > <Image src={venuegen} alt="VenueGen mostrando acciones para copiar o tomar luces y cámaras de VENUE. También hay un botón para generar y otro para generar y sobrescribir." />
* F10 - <a href="https://github.com/smcconne/Song-Progress-Tracker" target="_blank" rel="noopener noreferrer">[VenuePreview]</a>: Visualiza mas o menos como va tu VENUE. Muestra efectos de post-procesado, luces y cámara. Puedes ver los variantes de Guitarra/Bajo, Guitarra/Teclado y Bajo/Teclado.  
  > <Image src={venuepreview} alt="Venue Preview mostrando la cámara actual, efecto de post-procesado actual y luces actuales." />
* K - Crea una entrada (1, 2, 1-2-3-4). Tienes que tener el cursor en el segundo compás de la canción.
  * Recuerda de mapear el tempo de tu canción primero porque esto mueve todo hacia la derecha!
* Q - Crea un marcador de tempo de compás, depende de donde está el cursor.
* SHIFT+Q - Mueve la barra de compás más cercano al cursor mientras lo sostienes.
* [Editor de MIDI] SHIFT+L - Importa un archivo de letra y lo coloca en las notas elegidas.
* [Editor de MIDI] CTRL+SHIFT+Desplazar con ratón - Ajusta la velocidad de las notas. Demasiado útil para Guitarra/Bajo PRO.
* [Editor de MIDI] ALT+SHIFT+S - Divide las notas entre la cuadricula que estas usando. Es muy útil cuando si quieres crear notas semicorchea rápidamente.
* [Editor de MIDI] NUMPAD 9 - Ajusta la duración de las notas al tamaño de la cuadrícula. Ayuda mucho si quieres ajustar varias notas al mismo tiempo a un tamaño especifico, como reducir notas a una fusa (1/32).

## Otros plugins incluidos
Estos son otros plugins que están incluidos en la plantilla:
* <a href="https://github.com/noahbaxter/chartchotic" target="_blank" rel="noopener noreferrer">[ChartChotic]</a> - Un reemplazo para RBN Preview moderno y con código abierto. Esta funciona en varias plataformas así que los de Linux (y Mac) finalmente pueden ver lo que están charteando.  
Este plugin tiene varias funciones buenas, ¡así que explóralo todo lo que puedas! 
  > <Image src={chartchotic} alt="ChartChotic demostrando una pista de batería." />
* <a href="https://github.com/Deevun155/ProKeysPreview" target="_blank" rel="noopener noreferrer">[Pro Keys Preview]</a> - Un plugin que te ayuda ver tus charts de Teclado Pro. Muy útil para ver las limitaciones de rango y los movimientos!  
  > <Image src={prokeyspreview} alt="Pro Keys Preview mostrando la pista de Teclado Pro en experto." />

## Descargas adicionales
¡Otras cosas importantes que deberías descargar!
* <a href="https://nemosnautilus.com/magma/" target="_blank" rel="noopener noreferrer">[MAGMA]</a> - Una herramienta para compilar tu custom para poder meterla al juego. Sigue siendo usada y se ha mejorado por los años.  
* <a href="https://nemosnautilus.com/nautilus/" target="_blank" rel="noopener noreferrer">[Nautilus (antes conocido como C3 CON Tools)]</a> - Un programa que ayuda con varias cosas relacionadas a archivos de Rock Band. Tiene varias herramientas integradas para customs.  
* <a href="https://github.com/mtolly/onyx/releases" target="_blank" rel="noopener noreferrer">[Onyx]</a> - Un programa que necesitas si estás probando con RPCS3, ya que exporta en un formato de PS3 moderno.
* <a href="https://rentry.co/UVR-SW-Instructions" target="_blank" rel="noopener noreferrer">[Ultimate Vocal Remover GUI]</a> - Contrario al nombre, esto hace mucho mas que eliminar vocales. Esto es un programa que usa IA para separar canciones a partes individuales, ya que eso ayuda inmensamente cuando charteas. La versión en el enlace tiene la habilidad de importar modelos Roformer. Aquí hay modelos que yo uso y recomiendo:
  * Demucs:
    * htdemucs_ft - Separa a cuatro partes.
    * htdemucs_6s - Separa a 6 instrumentos en vez de 4.
  * MDX-NET
    * <a href="https://huggingface.co/jarredou/BS-ROFO-SW-Fixed/tree/main" target="_blank" rel="noopener noreferrer">[BS Roformer SW]</a> - Como htdemucs_6s pero en mejor calidad. Requiere una gráfica buena.
    * UVR-MDX-NET INST HQ 3 or 5 - Esencial porque separa vocales mejor que Demucs.
    * <a href="https://github.com/jarredou/models/releases/tag/aufr33-jarredou_MDX23C_DrumSep_model_v0.1" target="_blank" rel="noopener noreferrer">[Drumsep]</a> - Esencial porque separa pistas de batería a partes individuales. Ayuda demasiado para chartear batería.
* <a href="https://mvsep.com/en" target="_blank" rel="noopener noreferrer">[Mvsep]</a> - Es como UVR pero en vez de correr en tu computadora, corre en un servidor. Estas restringido a calidad de MP3 y hay cola. Puedes pagar para evitar esto.
