Esta es una guía para echar andar REAPER sin tener que buscar archivos en varios lugares.  
Todos los archivos incluidos están hechos para la versiones más actuales de REAPER y Python en Windows y Linux.

<Aside>¡Esta guía no te va a enseñar a chartear!</Aside>
Si necesitas chartear, hay otros lugares. Aquí esta una lista de recursos buenos para aprender a cheartear:

### Español
Tristemente, hay pocos recursos para aprender a chartear en Español por ahora. Estos son los que existen por ahora:

* Existe una lista de vídeos hechas por <a href="https://www.youtube.com/playlist?list=PLTaIifg4Ipx97Ao6KPb_yP5esRXq0cJ-O" target="_blank" rel="noopener noreferrer">[Pumpster Zone]</a>. Hay ciertos ajustes que no son relevantes ahora pero mayoría de los conceptos todavía sirven.

* La otra guía de Español es mas vieja y por <a href="https://www.youtube.com/playlist?list=PLBKqrSJanqQaJeIWi7tliYVn8q2lVjIQi" target="_blank" rel="noopener noreferrer">[Pipiris24]</a>. Estos vídeos tienen 10 años y obviamente la información no sera actual pero puede que hay unos consejos buenos.

### Ingles

#### Escritas
Si entiendes ingles (o puedes soportar usar el traductor), hay demasiados recursos.

* La <a href="http://docs.c3universe.com/rbndocs/index.php?title=Authoring" target="_blank" rel="noopener noreferrer">[guía de RBN/C3]</a> fue como aprendimos muchos. Esta guía fue escrita por Harmonix y tiene consejos adicionales del equipo de C3.
  * <a href="https://web.archive.org/web/20250417062932/http://docs.c3universe.com/rbndocs/index.php?title=Authoring" target="_blank" rel="noopener noreferrer">[Versión en Wayback por si acaso la versión principal se cae].</a>

* Hay <a href="https://keepitfishy.com/doa/CustomsBookv1.pdf" target="_blank" rel="noopener noreferrer">[un libro en formato PDF muy bueno que fue escrito]</a> por doa pero ciertas partes mencionan que instales una versión antigua de REAPER. Puedes y debes de ignorar eso.

* <a href="https://therogerland.tumblr.com/proguide" target="_blank" rel="noopener noreferrer">[Guía de Guitarra PRO]</a> - Probablemente, nunca vas a tocar Guitarra/Bajo PRO, pero para los masoquistas, aquí está la única guía buena para eso. Tristemente, solo está en Inglés pero la puedes traducir.

#### Vídeos
* Si eres mas de vídeos <a href="https://www.youtube.com/playlist?list=PLrIDLbHWXFJcidr8R95lH_gNfWpC-6hFh" target="_blank" rel="noopener noreferrer">[hay una lista de vídeos]</a> hecha por Broforce que usa esta guía como base de instalación.

* Para vocales, <a href="https://www.youtube.com/@addymillscustoms" target="_blank" rel="noopener noreferrer">[AddyMills]</a> hizo un vídeo que utiliza funciones que son exclusivas a las versiones de REAPER actuales.  
<a href="https://youtu.be/Vlq8NI2KOhg" target="_blank" rel="noopener noreferrer">[Pt. 1]</a>  <a href="https://youtu.be/whh5BYPx7Fs" target="_blank" rel="noopener noreferrer">[Pt. 2]</a>

* También existe <a href="https://www.youtube.com/watch?v=FHeGkA4nv4g" target="_blank" rel="noopener noreferrer">[un video meticuloso hecho por Truce]</a> que existe pero no es actual y hay veces que te dice que instales una versión de REAPER vieja. Como con el libro por doa, puedes y debes de ignorar esto.
