Con el tiempo, puede que ciertas canciones no salgan en tu colección de música. Esto usualmente puede pasar cuando instales varios paquetes con canciones duplicadas.
<Image src={rb3songcount} alt="Rock Band 3 mostrando una colección de música que le falta 5 canciones." />

Puedes usar Nautilus para resolver este problema.

<Aside>**Por favor nota que esto no es lo mismo que [[canciones que no salen porque tienen problemas de IDs]](../troubleshoot/#canciones-no-están-en-mi-colección-no-salen-en-línea--puntajes-no-se-guardan).**</Aside>

<Steps>
1. Abre Nautilus y elije <span title="Indexador de datos">"`File Indexer`"</span>.
    > <Image src={nautilushomefileindex} alt="Nautilus' main screen. A cursor hovers over 'Quick Pack Editor'." />

2. Haz click en <span title="Agregar nueva carpeta">"`Add New Folder`"</span>.
    > <Image src={nautilusfileindexaddfolder} alt="Dentro de File Indexer. El cursor esta sobre 'Add New Folder'." />

3. Ve a la carpeta donde están los datos (DLC) de Rock Band.  
Esto es `dev_hdd0\game\BLUS30463\` para contenido de Rock Band 3 y usualmente `dev_hdd0\game\BLUS30050\` para contenido de Rock Band 1/2.
    > <Image src={addfolder} alt="Un explorador de Windows. Esta elegida una carpeta de Rock Band 3 que se llama 'BLUS30463'." />

4. Activa <span title="Modo de PS3 (revisa por archivos de songs.dta)">"`PS3 Mode (check for songs.dta files)`"</span> en el medio y luego haz click en <span title="Crear índice">"`Build Index`"</span>.
    > <Image src={nautilusfileindexbuild} alt="Dentro de File Indexer con 'PS3 Mode (check for songs.dta files)' activo y el cursor sobre 'Build Index'." />

5. Haz click en <span title="Si">"`Yes`"</span> en la advertencia. Simplemente dice que puede tardar un rato, depende del tamaño de tu colección de música.
    > <Image src={nautilusfileindexyes} alt="Una advertencia que dice que puede tardar un rato. El cursor esta sobre 'Yes'." />

6. Cuando termine de escanear, entra a <span title="Herramientas">"`Tools`"</span> arriba y activa <span title="Buscar canciones con conflictos de ID">"`Find songs with ID conflicts`"</span>.
    > <Image src={nautilusfileindexselid} alt="Dentro de 'File Indexer' dentro del menu de 'Tools'. El cursor esta sobre 'Find songs with ID conflicts'." />

7. Tendrás una lista abajo que mostrara canciones duplicadas. Usualmente esto pasa por instalar varios paquetes con canciones duplicadas.
<Aside>Puede que tengas que ajustar el tamaño de la pantalla para ver toda la información.</Aside>
    > <Image src={nautilusfileindexlist} alt="Una lista de canciones en 'File Indexer'. Cada canción tiene una copia duplicada abajo porque la canción esta en dos paquetes." />

8. Haz click derecho en la canción (o canciones) que quieres borrar y elige <span title="Eliminar archivo(s) elegido(s)">"`Delete selected file(s)`"</span>. También puedes presionar la tecla de `Supr(imir)`.
    > <Image src={nautilusfileindexdeldup} alt="Una canción duplicada dentro de 'File Indexer'. Después de hacer click derecho en la canción duplicada, el cursor esta sobre 'Delete selected file(s).'." />
</Steps>
